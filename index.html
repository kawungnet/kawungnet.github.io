A robust network platform for covering high density area


Long range (LoRa) is a wireless RF technology which has an ability for sending amount of data in long communication range. Kawung network enhance LoRa ability to keep going well in high density area by using mesh network topology.

Kawung network is an open network and empowered by community


New node can join easily to an existing kawung network in a area. With new node join, it increase the network coverage. The network is fully maintained by the community in the area.

Your data is secured by end-to-end encryption


Kawung network use end-to-end encryption to make sure every data passing through the network is secured and can't read by other non-permissible node in public mesh network.

Kawung Network Technology aims to provide a secure LoRa public mesh network via Kawung Protocol. We encourage communities to collaborate with each other to build their IoT projects on a larger scale by supporting each others.

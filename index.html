A robust network platform for covering high density area


Long range (LoRa) is an wireless RF technology which has an ability for sending amount of data in long communication range. Dhandhanggula network have enhanced LoRa ability to keep going well in high density area by using mesh network topology.

Dhandhanggula network is an open network and empowered by community


New node can join easily to an existing dhandhanggula network in a area. With new node join, it increase the network coverage. The network is fully maintained by the community in the area.

Your data is secured by end-to-end encryption


Dhandhanggula network using AES encryption to make sure every data passing through the network is secure. We also working in another encryption algorithm to work in microcontroller ecosystem.
